這是一個能將任何網頁文章、選取文字或貼上的內容轉換為「英文書寫體（Cursive）」習字帖的 Chrome 擴充功能。支援匯出高品質 PDF，方便列印練習。

## 🌟 主要功能
- **右鍵轉換**：選取網頁上的任何一段英文，點擊滑鼠右鍵即可製作習字帖。
- **純文字轉換**：在插件 Popup 視窗中直接貼上文字進行轉換。
- **整頁抓取**：一鍵抓取當前網頁的所有內容並轉換為習字帖。
- **PDF Ready**：精心設計的四線格背景與書寫體字型，適合 A4 紙張列印。

## 🚀 安裝教學 (開發者模式)
1. 下載本專案並解壓縮。
2. 開啟 Chrome 瀏覽器，進入 `chrome://extensions/`。
3. 開啟右上角的 **「開發者模式 (Developer mode)」**。
4. 點擊 **「載入未封裝項目 (Load unpacked)」**。
5. 選擇本專案內的 `CursiveExtension` 資料夾。

## 🛠️ 技術說明
- **Manifest V3**：遵循最新的 Chrome 擴充功能規範。
- **CSP 規範**：JavaScript 完全分離，符合安全性要求。
- **Fonts**：使用 Google Fonts (Dancing Script)。
- **Layout**：純 Vanilla CSS 打造高效能渲染。

## 🔐 安全與隱私
- 所有資料轉換皆在瀏覽器本地完成，絕不將內容上傳至任何伺服器。
- 本專案採用 **Leon 鐵甲防護協議 (Iron Shield Protocol)**，確保環境變數與敏感資訊隔離。
